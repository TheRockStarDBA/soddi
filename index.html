SODDI - StackOverflow Data Dump Importer

StackOverflow Data Dump Importer. Gets the public data dump XML into a Microsoft SQL Server database.

Download this project as a .zip file Download this project as a tar.gz file

StackOverflow.com shares their database publicly.

For years, StackOverflow has been a great place to get your programming questions answered. Along the way, they've accumulated a ton of data - questions, answers, comments, users, votes, badges, and more. They provide all of that data in a public XML data dump updated a few times per year.

This application (SODDI) takes that XML data dump, and inserts it into a Microsoft SQL Server database. The result: a clean, simple, easy-to-understand database with real-world data that's fun to use for demos and learning SQL.
